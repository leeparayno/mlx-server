# Building MLX Server in Xcode

## Quick Setup

1. Open the package in Xcode:
   ```bash
   cd /Users/lee.parayno/code4/business/mlx-server
   xed .
   ```

2. In Xcode, click on the scheme selector (next to the Play/Stop buttons at the top)

3. Select "mlx-server" scheme from the dropdown

4. Change the run destination to "My Mac" (not an iOS device/simulator)

5. Press Cmd+B to build, or Cmd+R to run

## Available Schemes

- **mlx-server** - Main server executable
- **mlx-benchmark** - Benchmarking tool
- **MLXCore** - Core library

## Command Line Build (Alternative)

If Xcode gives you trouble, you can always build from command line:

```bash
swift build --product mlx-server
swift run mlx-server --help
```

## Troubleshooting

If you don't see schemes:
- Product → Scheme → Manage Schemes
- Make sure "mlx-server" is checked
- Make sure "Show" is enabled

If build fails with missing modules:
- Product → Clean Build Folder (Cmd+Shift+K)
- Close Xcode
- Run: rm -rf ~/Library/Developer/Xcode/DerivedData/mlx-server-*
- Reopen Xcode
